# Scala MX
A collection of libraries for full stack Scala matrix.

+ JDK
  + 8
  + 11
+ Scala
  + 0.22.0-RC1
  + 2.13.1 (default)
  + 2.12.10
  
## Libraries
+ Guava re-implemented in Scala
  - guava-base
  - guava-collect
  - guava-concurrent
  - guava-primitives
  
## Switching Scala Versions
```
sbt -Dscala.version=0.22.0-RC1
```
